## 🔄 Azioni Base (Appunti e Cronologia)
* **Ctrl + C / Ctrl + V** : Copia e Incolla
* **Ctrl + X** : Taglia
* **Ctrl + Z** : Undo (annulla l'ultima operazione fatta)
* **Ctrl + Y** : Redo (annulla l'ultimo Ctrl+Z)

## 🖱️ Selezione del Testo
* **Doppio click sinistro** : Seleziona l'intera parola
* **Triplo click sinistro** : Seleziona l'intera riga
* **Maiusc + Freccette** : Seleziona il testo/codice lettera per lettera
* **Ctrl + Maiusc + Freccette** : Seleziona il testo saltando di parola in parola
* **Ctrl + A** : Seleziona tutto il testo presente nel file

## ↕️ Gestione e Spostamento Righe
* **Ctrl + D** : Cancella l'intera riga corrente
* **Alt + Ctrl + Freccia Giù** : Duplica la riga corrente
* **Alt + Freccia Su / Giù** : Sposta la riga su o giù nel file
* **Shift + Enter** : Va a capo in una nuova riga senza spezzare il testo della riga in cui ti trovi

## ❌ Cancellazione Rapida
* **Ctrl + Canc / Backspace** : Cancella il testo "a botte di parole" (invece che singole lettere)

## 🔍 Ricerca
* **Ctrl + F** : Trova (e sostituisci) all'interno del file

## 💻 Scrittura Codice e Formattazione
* **Tab / Shift + Tab** : Aumenta o riduce l'indentazione (funziona anche selezionando più righe contemporaneamente)
* **Ctrl + /** *(tasto 7 o Shift+7)* : Commenta o decommenta la riga
* **sout + Tab** : Genera in automatico il comando `println`
* **Delimitatori** : Ricorda l'uso delle coppie per le chiusure automatiche `( " , ( , { , [ )`
